1) Let n = str1.length, m = str2.length, create a 2D array dp[n+1][m+1]

2) Fill dp using two loops:
     if str1[i-1] == str2[j-1] then. dp[i][j] = 1 + dp[i-1][j-1]
     else  dp[i][j] = max(dp[i-1][j], dp[i][j-1])

3) Build the LCS string:
     start from i = n, j = m, empty string lcs
     if characters match then, add to lcs, move diagonally (i--, j--)
     else move in direction of larger dp value
     reverse lcs at the end

4) Initialize pointers i = 0, j = 0, and empty string result

5) Traverse each character c in lcs:
     while str1[i] != c ,append str1[i], i++
     while str2[j] != c ,append str2[j], j++
     append c, then increment both i++, j++

6) Append remaining characters of str1 from i to end
7) Append remaining characters of str2 from j to end
8) Return result
